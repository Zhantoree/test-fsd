## Folders description

| Folder       | Description        |
| ------------ | ------------------ |
| public       | Public files       |
| src/app      | App FSD Layer      |
| src/pages    | Pages FSD Layer    |
| src/widgets  | Widgets FSD Layer  |
| src/features | Features FSD Layer |
| src/entities | Entities FSD Layer |
| src/shared   | Shared FSD Layer   |



Скачиваем библиотеки
```bash
    npm install
```

Запуск командой
```bash
    npm run dev
```
